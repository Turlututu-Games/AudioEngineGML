# Game Sound

Game type of sound are designed to not run in loop, and to not accross room. It mainly can be used for diegetic sounds. 

Game sound can also be spatialized, to correspond as an object position

Like for UI sound, Game sound can be configured to randomize the volume or the pitch, to allow some variety.

It also possible to configure an array of sound assets. In this scenario, a random sound from this array will be selected each time
